# MOCE 产品原型复刻

基于 `PRD-images/` 中 13 张产品原型图的 **1:1 高保真 Web 复刻**。

## 快速开始

```bash
cd web
bun install
bun dev
```

浏览器打开终端提示的本地地址（默认 `http://localhost:5173`）。

## 目录

```
moce/
├── PRD-images/     # 原始原型图 (1.png … 13.png)
├── web/            # Bun + Vite + React 实现
└── README.md
```

详见 [web/README.md](./web/README.md)。
