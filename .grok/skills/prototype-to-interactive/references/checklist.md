# 交付验收清单

## 视觉

- [ ] 主布局对齐对应原型图（顶栏、主区、步骤条）
- [ ] 配色 / 圆角 / 阴影与截图一致
- [ ] 背景动效存在（粒子/光晕/网络）
- [ ] 宽屏（≥1024）左右贴画可见、垂直居中、半透明
- [ ] 贴画在多帧间轮询，窄屏有 strip 或可接受降级

## 交互

- [ ] 顶栏：导航、登录、开始使用、主题切换均有效
- [ ] 主 CTA 进入正确流程页
- [ ] 次要按钮有 Toast / Modal / 下载之一
- [ ] 筛选/因子：改下拉不立即改数据；点应用后改
- [ ] 图表类型切换主图变化
- [ ] 图表配置（X/粒度）标签与点数变化可见
- [ ] 列表/节点选中 → 详情区全量更新
- [ ] 多步向导每步内容不同
- [ ] 跨页状态（查询文案等）保持

## 深色

- [ ] 切换后背景、卡片、文字、输入框统一变暗
- [ ] 刷新后主题保持

## 工程

- [ ] `bun run build` 通过
- [ ] README 含本地与线上地址
- [ ] `docs/PRODUCT.md` + 可选 `PITFALLS.md`
- [ ] Git 提交；需要时 Vercel 生产部署
