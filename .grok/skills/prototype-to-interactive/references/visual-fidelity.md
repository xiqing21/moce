# 视觉保真：识图 ≠ 最像

完整论述见项目文档 `docs/VISUAL-FIDELITY.md`（MOCE 仓库）。

## 一句话

**核心能力是：从原型抽取可执行设计规格（token / 布局拓扑 / 层级）并用代码对照原图迭代——不是「更能识图」或「更能文生图」。**

## 做 1:1 时强制

1. 先抽 design token，再写页面  
2. 画布局拓扑（贴画 / 主区 / 字阶）  
3. 抑制默认 SaaS 模板脸  
4. 贴画 Grid + 低透明 + 居中  
5. 多轮并排原图 vs 实现做 diff  

## 常见失败

- 语义对、气质错（通用 AI UI）  
- 图像→文字摘要过粗再写码  
- 一次生成不迭代  
- 只聊天贴代码，无工程闭环
